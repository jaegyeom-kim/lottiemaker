# LottieMaker

로티(Lottie) 애니메이션을 빠르고 쉽게 만들고 수정하는 웹 도구.
After Effects 없이 — 템플릿에서 시작해 픽셀 단위로 커스텀하고, 기존 JSON도 열어서 편집합니다.

## 실행

```bash
npm install
npm run dev   # http://localhost:5173
```

## 기능

- **템플릿 갤러리 15종** — 로딩 6 / 피드백 2 / 인터랙션 5 / 이펙트 2, 실시간 미니 프리뷰
- **템플릿 옵션** — 템플릿마다 2~3개의 파라메트릭 슬라이더, 전부 픽셀(px)·도(°) 절대 단위
  - 예: 스피너(링 지름·선 두께·아크 각도), 웨이브 바(바 너비·최대 높이), 컨페티(퍼짐·회전량·조각 크기)
- **실시간 미리보기** — 재생/일시정지, 프레임 스크럽, 배속(0.25–2x), 루프, 배경 전환
- **사용 예시 목업** — 폰 프레임 안 실제 서비스 맥락 5종(다이얼로그/버튼/엠티/리스트/토스트)에 배치해 확인
- **편집** — 단색 fill/stroke 색상 일괄 변경, 캔버스 크기, 레이어 표시/숨김, 실행 취소/다시 실행(⌘Z)
- **가져오기/내보내기** — JSON 드래그앤드롭, 다운로드, 클립보드 복사, 배속 굽기 옵션

## 스택

React 19 · TypeScript · Vite · zustand · lottie-web

## 구조

```
src/
  lib/lottieKnobs.ts    # 파라메트릭 옵션 엔진 (줌/두께/트림/진폭/도형 크기, px·° 단위)
  lib/lottieColors.ts   # 색상 추출/치환 (fill·stroke, 정적+키프레임 s/e)
  lib/lottieUtils.ts    # 파싱/검증, 레이어, 리사이즈, 배속 굽기, 다운로드
  store.ts              # zustand 스토어 + 스냅샷 undo/redo (노브 상태 포함)
  components/           # 프리뷰, 목업, 갤러리, 옵션/색상/레이어/내보내기 패널
  templates/            # 내장 로티 JSON 15종 + 노브 정의 레지스트리
```

## 옵션 동작 방식

노브는 항상 노브 미적용 원본에서 전체 재계산돼 값이 누적 왜곡되지 않습니다.
진폭 계열(바운스/퍼짐/흔들림)은 첫 키프레임 기준 편차를 조절해 순환 루프가 깨지지 않고,
크기 계열은 캔버스 중심 기준 줌이라 잘림 없이 구도 전체가 움직입니다.
