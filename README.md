# LottieMaker

로티(Lottie) 애니메이션을 빠르고 쉽게 만드는 웹 도구.
After Effects 없이 — 템플릿에서 시작하거나, 내 그래픽(SVG/PNG)을 올려 모션을 조립하고, 결과를 Lottie JSON으로 내보냅니다.

**라이브:** https://jaegyeom-kim.github.io/lottiemaker/

## 실행

```bash
npm install
npm run dev   # http://localhost:5173
```

## 두 가지 모드

왼쪽 사이드바 탭이 곧 작업 모드입니다. 탭·캔버스·우측 패널·타임라인이 항상 한 모드로 움직이고,
탭을 전환하면 작업공간이 통째로 스왑되므로(실행 취소 히스토리 포함) 어느 쪽 작업도 사라지지 않습니다.

### 템플릿 모드

- **갤러리 25종** — 로딩 7 · 피드백 3 · 인터랙션 9 · 이펙트 6. 호버 시 재생, 검색, 최근 사용, 현재 열린 템플릿 표시
- **파라메트릭 옵션** — 템플릿별 슬라이더(픽셀·도 절대 단위) + 직접 수치 입력, 토글, 선택 칩, 재생 길이
- **폰트** — 숫자 카운트 등 텍스트 템플릿은 프리셋 + 로컬 폰트(Local Font Access API, Chromium) 선택
- **그래픽 교체** — 지정 슬롯(벨, 점, 핀 등)에 SVG/PNG를 넣어 내 그래픽으로 교체, 마우스로 앵커 조정
- **색상 일괄 변경** — 문서의 단색 fill/stroke를 그룹으로 추출해 한 번에 교체
- **사용 예시 목업** — 폰 프레임 속 실제 서비스 맥락(다이얼로그/버튼/리스트/토스트 등)에 배치해 확인

### 커스텀 모드

- **그래픽 업로드** — SVG/PNG/JPG/WebP 여러 장, 드래그앤드롭(드롭 지점 배치) 또는 패널 업로드. SVG는 벡터 셰이프로 변환
- **등장 / 루프 / 퇴장 3슬롯 모션** — 레이어마다 조합
  - 등장: 페이드 · 아래/위/왼쪽/오른쪽에서 · 팝 · 드롭 (+오버슛 바운스, 딜레이, 시간, 거리)
  - 루프: 플로팅 · 펄스 · 흔들기 · 회전 · 바운스 (세기·주기 — 클립에 정수 사이클로 맞춰져 끊김 없음)
  - 퇴장: 페이드 · 아래/위/왼쪽/오른쪽으로 · 축소
- **타임라인 (AE 컴포지션 방식)** — 레이어별 클립을 드래그로 시간 이동·트림, 등장(초록)/퇴장(빨강) 세그먼트를 잡아 길이 조절, 스냅(사분점·플레이헤드·다른 클립), 플레이헤드 드래그·스크럽
- **캔버스 편집** — 이동(V)/핸드(H) 툴, ⌘+휠 줌·휠 팬, 스냅 가이드(중앙·사분점·모서리), 리사이즈 핸들(기본 반대 모서리 고정 · Alt 중앙 기준 · Shift 10px 스냅), Alt+드래그 복제, 다중 선택(Shift/⌘ 클릭), 빈 곳 클릭 = 선택 해제
- **정렬 패널** — 일러스트레이터식 정렬 6종 + 균등 분배, 기준은 캔버스 또는 선택 영역
- **앵커 포인트** — 선택 박스의 ⌖ 버튼으로 팝업, 9점 그리드 + 드래그 패드. 그래픽은 제자리(팬 비하인드), 회전·스케일 피벗만 변경
- **컴포지션 길이** — 0.5~6초. AE 컴프처럼 늘려도 각 레이어의 애니메이션 타이밍은 절대 시간 유지
- **레이어 패널** — 이름 변경(더블클릭), 표시/숨김, 복제, 삭제, ≡ 드래그로 순서 변경, 레이어 라벨 컬러

## 저장

- **자동 저장** — 편집 멈추고 0.8초 후 localStorage에 기록. 템플릿/커스텀 슬롯 분리, 새로고침·창 닫기 직전에도 플러시. 상단바 배지가 실제 저장 결과를 표시(용량 초과 시 경고)
- **프로젝트 파일 (.lmproj.json)** — 에셋 내장(래스터 base64 + SVG 원문)으로 파일 하나로 완결. 미리보기에 드롭하면 이어서 편집
- **예제** — [`examples/커스텀-예제.lmproj.json`](examples/커스텀-예제.lmproj.json) 을 드롭해 커스텀 기능 전반(3슬롯 모션 · 스태거 · 클립 오프셋 · 앵커 피벗)을 바로 확인

## 내보내기

Lottie JSON 다운로드 / 클립보드 복사 (배속 굽기 옵션), 프로젝트 파일 저장.
텍스트 템플릿은 사용 폰트 힌트를 함께 표시합니다.

## 테마

다크 / 라이트 / 시스템 설정 따라가기(실시간 반영) — 상단바 버튼으로 순환, 선택 유지.

## 단축키

| 키 | 동작 |
|---|---|
| `Space` | 재생/일시정지 |
| `V` / `H` | 이동 툴 / 핸드 툴 |
| `⌘Z` / `⇧⌘Z` | 실행 취소 / 다시 실행 |
| `⌘D` | 레이어 복제 |
| `Delete` | 선택 레이어 삭제 |
| `방향키` (+`Shift`) | 1px (10px) 이동 |
| `Esc` | 드래그·리사이즈 취소, 팝업 닫기 |
| `⌘0` | 100% / 중앙 |
| `⌘+휠` / `휠` | 줌 / 팬 |
| 드래그 중 `Shift` / `Alt` / `⌘` | 축 고정 / 복제 / 스냅 해제 |
| 리사이즈 중 `Alt` / `Shift` | 중앙 기준 / 10px 스냅 |

## 구조

```
src/
  lib/customBuilder.ts   # 커스텀 엔진 — 3슬롯(xsel) → 키프레임 생성, 클립/앵커/라벨 컬러
  lib/svgImport.ts       # SVG → 로티 셰이프 변환, 래스터 임베드
  lib/lottieKnobs.ts     # 템플릿 파라메트릭 옵션 엔진 (줌/두께/진폭/개수 등, px·° 단위)
  lib/lottieColors.ts    # 색상 추출/치환 (fill·stroke, 정적+키프레임)
  lib/lottieUtils.ts     # 파싱/검증, 레이어, 리사이즈, 배속 굽기, 다운로드
  store.ts               # zustand 스토어 — 모드 스왑, 스냅샷 undo/redo, 자동 저장
  components/            # 프리뷰(캔버스·타임라인), 갤러리, 커스텀 빌더, 정렬/색상/레이어/내보내기 패널
  templates/             # 내장 로티 JSON 25종 + 노브 정의 레지스트리
examples/                # 커스텀 데모 프로젝트 파일
```

## 동작 원리 메모

- **노브**는 항상 노브 미적용 원본에서 전체 재계산 — 값이 누적 왜곡되지 않습니다. 진폭 계열은 첫 키프레임 기준 편차 조절이라 순환 루프가 깨지지 않습니다.
- **커스텀 레이어**는 표준 로티 문서에 확장 필드(`xsel` 슬롯 상태, `xbase` 정착 위치, `xci` 라벨 컬러, `xsrc` SVG 원문)를 얹은 구조 — 로티 플레이어는 이를 무시하므로 내보낸 JSON은 어디서든 재생됩니다. 키프레임은 편집 때마다 `xsel`에서 재생성됩니다.
- **컴포지션 길이** 변경은 `op`만 바꾸고 클립·키프레임은 절대 프레임을 유지 — AE 컴프 길이 변경과 같은 의미론입니다.
